# JavaScript Course - GitHub Ready

This folder is prepared so you can upload all your JavaScript learning work to GitHub in one place.

## What is included

- `index.html` -> New modern portfolio homepage
- `assets/` -> Portfolio styles, JavaScript, and images
- `source-files/` -> Your original JavaScript course files and projects
- `source-files/clean-projects/` -> Clean copy of car-selling project for direct GitHub use

## How to use

1. Open `index.html` in browser to view your new portfolio.
2. Open project links from the portfolio cards.
3. Upload the complete `JavaScript-Course-GitHub-Ready` folder to GitHub.

## Notes

- Your old files are preserved in `source-files`.
- Portfolio design includes animations, counters, and project filters.
- `.gitignore` already excludes one embedded nested repository from copied legacy files.
